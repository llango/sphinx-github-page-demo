欢迎使用Sphinx Github Page Demo¶

先克隆下来，git clone https://github.com/llango/sphinx-github-page-demo.git, 将rst文档放在source。 make html cp -a _build/html/. ./docs

其中你可以使用sphinx创建的文档，如果要使用shaphinx主题，可以修改conf.py 里面html_theme 的值。 然后创建docs，这个是便于github pages 选择这个目录操作。 在docs目录创建一个.nojekyll文件。
